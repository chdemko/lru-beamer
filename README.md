# Beamer template for La Rochelle University

Usage:

~~~latex
\usetheme{LaRochelle}
\usepackage{raleway} % or verdana, DejaVuSans
~~~

or use this option:

~~~latex
\usetheme[pagenumber]{LaRochelle}
\usepackage{raleway} % or verdana, DejaVuSans
~~~

The output of `example.tex` gives the `example.pdf` file.
